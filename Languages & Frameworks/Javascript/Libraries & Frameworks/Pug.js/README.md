* Pug is a template engine.
* It can be used with frontend frameworks like Vue.js.
* Tries to simplify html templates.

# Quoto
* Can be any valid quoto character.
  * Single
  * Double
  * Backtick

# Attributes
```pug
a(href='google.com' class='btn')
input(
    type='text'
    class='input'
    name='test'
    id='input-test'
)
```
